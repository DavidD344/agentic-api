Descoberta de Padrões para Redução do Consumo de Combustível em Locomotivas ? PR2C-L
Descrição: O objetivo deste projeto é descobrir padrões para a redução do consumo de combustível em locomotivas. Tais também devem contribuir para uma condução segura de uma composição formada por uma ou mais locomotivas e diferentes vagões. A consecução deste objetivo passará pela aplicação de técnicas de aprendizagem de máquina e descoberta de conhecimento a partir de conjuntos de dados de viagens de trens que transportam cargas. Os conhecimentos descobertos serão usados para dotar um agente de condução de trem, denominado de CARGO. O comportamento deste agente consistirá em executar planos de ações, cujas ações possíveis são: aumentar, manter ou reduzir velocidade. O agente CARGO deverá identificar os pontos de aceleração que gerarão os menores consumos para uma viagem. A hipótese para uma boa condução é realizar uma viagem mantendo, por mais tempo possível, o mesmo ponto de aceleração, bom como velocidade constante e próxima da máxima para o trecho..
