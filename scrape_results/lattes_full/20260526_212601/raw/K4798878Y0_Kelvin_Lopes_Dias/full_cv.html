Pesquisa e Desenvolvimento em Open RAN e Núcleo 6G Nativo de IA
Descrição: Este projeto envolve três frentes em direção às redes móveis de próxima geração (6G) nativas de IA (Inteligência Artificial): 1) Suporte a interfaces abertas, programabilidade e rApps/xApps da arquitetura Open RAN para o gerenciamento de mobilidade, monitoramento, redirecionamento de tráfego e garantias de QoS; 2 ) Segment Routing na rede de transporte e 3) Núcleo 5G/6G nativo de IA com evolução do NWDAF (Network Data Analytics Function) e emprego de IA agêntica. Os primeiros resultados das pesquisas relacionadas a este projeto foram publicados em eventos nacionais (SBRC e SBrT) e internacionais (IEEE)..
Situação: Em andamento; Natureza: Pesquisa.

2024 - Atual
Gerenciamento de Mobilidade em Redes 5G/6G Abertas e Definidas por Software
Descrição: As futuras redes de sexta geração das comunicações móveis (6G Sixth Generation) devem proporcionar experiências e serviços cujos requisitos ainda não serão contemplados pela atual quinta geração (5G - Fifth Generation), tais como comunicação holográfica, gêmeos digitais e experiências imersivas baseadas em realidade estendida, maior eficiência energética e conectividade ubíqua. Para além dos avanços e tecnologias da nova interface aérea 6G que ainda será definida, a indústria e a academia estão considerando três pilares para a concepção de uma arquitetura 6G virtualizada e definida por software para prover suporte aos requisitos e aplicações preconizadas para a nova geração de redes celulares. Primeiramente, o emprego massivo de computação de borda para processamento e armazenamento mais próximos do usuário e de objetos da Internet de Tudo (IoE - Internet of Everything). O segundo pilar consiste em arquiteturas com interfaces abertas para promover inovação e programabilidade na borda e no núcleo da rede, com a participação de todo o ecossistema de TIC e não apenas de poucos fornecedores de infraestrutura. P4Sec - Securing Networks in the Programmable Data Plane Era
Descrição: Recent advances in Software Defined Networking (SDN) have expanded our ability to program the network to its data plane. Through domain specific languages like P4 [3], network operators can quickly deploy new protocols on forwarding devices, customize their functionality, and develop innovative services [13, 5, 17]. This flexibility comes however with a cost: network-wide security and correctness properties (e.g., isolation, reachability, etc.) become much harder to ensure, because network behavior is now determined by a combination of the control plane-driven configuration and the data plane program that resides on devices (also called switches). Existing network verification tools [7, 14, 19, 15, 35], which rely on a fixed, invariant model of the data plane, are inadequate for programmable data planes. We propose to develop new techniques to verify and enforce security properties in data plane networks. Our proposed verification techniques extend existing verification tools by automatically generating a data plane model from a P4 program.We then adapt existing verifications tools to work with our dynamicallygenerated models to verify network configuration updates issued by an SDN controller. We also propose a new approach to ensuring that network security properties are satisfied by a network configuration that is based on data plane enforcement. We propose to develop an in-line monitor, implemented in the data plane itself, that enforces critical security properties, such as isolation and bandwidth limits, even in the presence of a faulty user data plane program or controller. Today?s networks are increasingly called on to perform security services, e.g. intrusion detection, in the network. A programmable data plane network can be used to accelerate and enhance such services by performing some of the required processing in the network data plane. The second thrust of this proposal concerns new data plane security services made possible by programmable data plane networks. We propose to develop new data plane-based security functions and techniques for composing them in a manner that balances performance characteristics of the data plane (very fast but less complex) and dedicated virtualized network function units attached to the network (slower but more complex) to implement the required security function. Programmable data planes are poised to transform the architecture of modern networks. The proposed work will advance the current state-of-the-art by extending verification and enforcement techniques to programmable data plane networks, for which neither network verification nor security policy mechanisms currently exist. Our work will also develop new data plane services that will enable a new class of security functions to be deployed in the network in order to improve the overall security of computer networks. Our project follows two branches of investigation: First, we propose to determine which network security properties can be verified in networks with programmable data planes and which properties can be dynamically enforced by a programmable data plane. Second, we investigate new security services made possible by programmable data planes..

2011 - 2013
SIMTUR - Sistema Inteligente de Monitoramento de Tráfego Urbano
Descrição: Questões recorrentes, tais como os grandes congestionamentos nos centros urbanos e a ausência de fomento à utilização de transportes coletivos, são obstáculos ao desenvolvimento de cidades inteligentes. Como soluções a essas dificuldades e visando atingir uma mobilidade sustentável, as TICs (Tecnologias da Informação e Comunicação) vêm sendo aplicadas, tendo como principal resultado os Sistemas de Transportes Inteligentes (STI). Neste contexto, este projeto propõe criar, modelar e implementar um sistema integrado de soluções, intitulado SIMTUR, direcionado ao rastreamento, comunicação, controle e monitoramento de transporte e tráfego em cidades. SIMTUR pretende prover informações sobre o tráfego, localização e demais condições urbanas (nível de poluição, temperatura, umidade e outras) aos usuários de transporte coletivo e privado, além de informações de itinerários para os passageiros de transportes coletivos. O projeto desenvolverá, ainda, um simulador de semáforos inteligentes e disponibilizará uma base de dados, que podem ser utilizados por instituições governamentais e prefeituras para auxiliar no planejamento urbano. Em geral, o sistema SIMTUR contribuirá para a desobstrução de vias urbanas e almeja incentivar o uso de transportes coletivos de mobilidade.
Situação: Concluído; Natureza: Pesquisa.
