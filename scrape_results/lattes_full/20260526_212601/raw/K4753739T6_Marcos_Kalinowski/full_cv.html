CNPq Universal 2024: AI Engineering - Engenharia de Software para Ciência de Dados e Inteligência Artificial
Descrição: Sistemas de software que envolvem inteligência artificial (IA) têm gerado benefícios expressivos para a sociedade e para os negócios, mas também impõem desafios inéditos à engenharia de software. Este projeto tem como objetivo desenvolver métodos, técnicas e ferramentas que viabilizem a construção de sistemas de IA com qualidade, confiabilidade e sustentabilidade. A proposta abrange múltiplos paradigmas de IA, incluindo machine learning, reinforcement learning, foundation models, agentic AI e symbolic AI, cujas características exigem adaptações e inovações nos processos tradicionais de engenharia de software. Adotando uma abordagem experimental, aplicada tanto na academia quanto na indústria, o projeto busca gerar soluções para especificação, gestão, arquitetura e controle de qualidade de sistemas de IA, promovendo avanços de alto impacto científico e prático..
Situação: Em andamento; Natureza: Pesquisa.

CNPq PQ - Engenharia de Software para Ciência de Dados e Inteligência Artificial
Descrição: Sistemas de software inteligentes envolvendo técnicas de ciência de dados e inteligência artificial estão se tornando cada vez mais comuns e se mostraram capazes de trazer benefícios para a sociedade e retornos significativos para diversas áreas de negócio. Entretanto, muitas vezes o que sabemos sobre software convencional não se aplica diretamente a sistemas de software inteligentes e as práticas da engenharia de software necessitam de adaptações para que tais sistemas possam ser construídos com qualidade. Considerando isto, a Engenharia de Software para Sistemas Inteligentes foi recentemente incluída como uma das seis áreas estratégicas da agenda nacional prioritária americana de pesquisa em engenharia de software. O presente projeto tem como objetivo ajudar a posicionar a pesquisa brasileira na vanguarda desta área reconhecida internacionalmente como estratégica, concebendo métodos, técnicas e ferramentas para auxiliar na construção de sistemas inteligentes com qualidade..

CNPq PQ - Identificação e Prevenção de Problemas Críticos da Engenharia de Requisitos
Descrição: [Contexto] Problemas da Engenharia de Requisitos (ER) podem levar projetos de software a serias consequências durante o ciclo de vida de desenvolvimento, incluindo o fracasso do projeto como um todo. [Objetivo] O principal objetivo deste projeto é compreender tendências e problemas da ER e propor soluções e tecnologias para prevenir a ocorrência destes problemas na indústria de software. Este objetivo envolve esforços no contexto da iniciativa NaPiRE (Naming the Pain in Requirements Engineering), coordenado pelo proponente juntamente com pesquisadores da Alemanha e Áustria.
