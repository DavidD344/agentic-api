3.
Enterprise Application Integration (EAI)

Objetivo: O objetivo desta linha de pesquisa é integrar um conjunto heterogêneo de aplicações de forma que as mesmas possam ser mantidas em sincronía ou colaborar para proporcionar novas funcionalidades com base nas já existentes. Nossas restrições ao criar estas soluções de integração são: 1) As aplicações atuais não deverão ser alteradas para serem integradas; 2) Não se adiciona acoplamento às aplicações que participam de uma solução de integração. A orquestração das aplicações é realizada de forma exógena pela solução de integração. Trabalhamos em uma linguagem específica de domínio (DSL) para modelar estas soluções de integração a um ato nível de abstração, independente de tecnologia. Descrição: O desenvolvimento e a migração de aplicações para um contexto de computação em nuvem têm se mostrado uma tendência crescente e irreversível nos últimos anos.
