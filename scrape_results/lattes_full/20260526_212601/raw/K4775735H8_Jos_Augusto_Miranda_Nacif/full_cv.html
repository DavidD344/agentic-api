GT-SmartSeg - Grupo de Trabalho em Segurança de Contratos Inteligentes
Descrição: Auditar segurança de contratos inteligentes é essencial para evitar vulnerabilidades em aplicações descentralizadas (DApps) em redes como Ethereum e Hyperledger Fabric. Embora existam ferramentas para detectar vulnerabilidades, a adoção dessas soluções enfrenta desafios, como projetos descontinuados ou de código fechado, e a necessidade de configurar diferentes ambientes de execução. A proposta visa desenvolver um framework automatizado para execução de ferramentas de verificação de segurança em contratos inteligentes, integrando auditorias de segurança ao ciclo de desenvolvimento de DApps. O framework será integrado à RNP, compatível com redes blockchain como Hyperledger Besu e Ethereum, e incluirá atualizações para detectar novas vulnerabilidades por meio de Fuzzing. Os principais benefícios incluem a automação da auditoria, redução de custos com auditorias tradicionais, e a disponibilização de ferramentas de verificação acessíveis. O sistema será acessado por uma interface web, gerenciando containers configurados para executar diversas ferramentas, e poderá ser adaptado a diferentes redes. O projeto também fornecerá documentação, vídeos demonstrativos e relatórios de desempenho..

Descrição: Edital CNPq/AWS 032/2019. Porém, para simular todo espaço de estados de uma GRN com n nós são necessários 2n passos, limitando os estudos a GRNs com poucos genes (da ordem de 20 a 40). Existem vários modelos com mais de 50 genes documentados na literatura que foram poucos explorados em função da sua complexidade. Este trabalho propõe o desenvolvimento de novas ferramentas e técnicas para facilitar o estudo de GRNs utilizando computação de alto desempenho de forma eficiente, escalável e transparente para usuário final. Dessa forma, será possível diminuir o tempo de execução de simulações em até 4 ordens de grandeza, permitindo a exploração de redes reguladoras maiores. As ferramentas podem ser usadas em computadores na nuvem com aceleradores (GPUs e FPGAs), explorando o paralelismo de grão grosso/fino, além de poderem ser integradas a outros softwares de ampla utilização na área de biologia como o Cytoscape. As ferramentas desenvolvidas neste projeto têm como objetivos específicos permitir que o usuário possa dinamicamente construir e ajustar os modelos de GRNs em tempo de execução. Sistemas Computacionais para Inovação em Sensores de Alta Tecnologia
Descrição: Edital Universal FAPEMIG 01/2016. Este projeto tem o objetivo de desenvolver sistemas computacionais para aplicações de sensoriamento de alta tecnologia. Serão desenvolvidos/finalizados 3 sistemas de sensoriamento para as aplicações: a) Qualidade de água de reservatórios: O sistema de monitoramento é capaz de monitorar temperatura, condutividade elétrica e oxigênio dissolvido a partir de uma sonda aquática; b) Irrigação inteligente: Utilização de um sensor de temperatura infra-vermelho para medir a temperatura das folhas que aumentam quando a planta precisa de irrigação; c) Identificação de poluição: Pretende-se utilizar fluorescência para estimar a quantidade de fotossíntese que uma planta está realizando.
